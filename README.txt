# Base Wallet Tracker

A simple web app to track wallet balances on the Base network.

## 🚀 Features
- Enter any wallet address
- Fetch ETH balance from Base
- Clean and simple UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Base RPC

## 📦 How to Run
1. Clone this repo
2. Open `index.html` in your browser

## 🔮 Future Improvements
- Show transaction history
- Add token balances
- Add wallet connection (MetaMask)
- Improve UI
