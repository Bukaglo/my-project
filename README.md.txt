# Base Smart Contract Checker

This project shows how to connect to the Base blockchain and read data from a smart contract.

## Features
- Connects to Base network
- Reads contract data (e.g., name, balance)
- Beginner-friendly

## Setup

1. Install dependencies:
   npm install

2. Add your RPC URL in `.env`:
   BASE_RPC_URL=your_rpc_url_here

3. Run:
   node index.js

## Example
Reads a token name or wallet balance from a contract.
